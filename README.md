Mold for (almost) everything, even for a mold!

## Get start

### Installation

Install with `npm`:

`npm install moldmold -D`

or with `yarn`:

`yarn add moldmld -D`

### Add a custom mold

`npx moldmold mold`

### Bake a mold

`npx moldmold bake`
